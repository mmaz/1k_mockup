1000 Words in 1000 Languages

Traditional datasets for keyword spotting require thousands of recorded utterances. We automate dataset generation for keyword-spotting for multiple languages using existing speech corpora, and apply few-shot methods for custom keyword detection. We aim for a fully automated, rapid time-to-solution pipeline for generating voice-based command interfaces for arbitrary keywords in any language, with suitability for deployment on low-cost, power-efficient microcontrollers.
